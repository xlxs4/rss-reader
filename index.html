Whether the system will recover from a particular finite-time disturbance to a desired stable equilibrium point depends on uncertain and time-varying system parameter values. Therefore, it is valuable to determine, for specific fixed disturbances, the margins for safe operation: the smallest change in parameter values that would cause the system to become vulnerable to the disturbance.
