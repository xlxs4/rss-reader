    The Cartesian Grid Active Flux Method with Adaptive Mesh Refinement
    Abstract We present the first implementation of the Active Flux method on adaptively refined Cartesian grids. The resulting method has a compact stencil in space and time and good stability properties. On each Cartesian grid patch the single grid Active Flux method can be applied. The exchange of data between grid patches is organised via ghost cells.
